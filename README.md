# Halo

Halo is a fake multi-tenant SaaS project/task management app built for Pendo experimentation and demos.

## Run

```bash
npm install
npm run dev
```

Opens at http://localhost:5173.

## Build

```bash
npm run build
npm run preview
```

## Phase

This is the Phase 1 scaffold. Mantine theming, routing, storage, providers, and primitives
are added in subsequent plans. See `.planning/ROADMAP.md` for the full build plan.
